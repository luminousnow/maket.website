Manage Leads list easier for Flamingo & Contact Form 7: Add the column to show field form in Inbound Messages list, Change Flamingo menu name (Ex: Flamingo » Contact Leads) etc.

== Description ==

This simple but powerful plugin will help your improve using Flamingo and Contact Form 7.

= Featured of Advanced Flamingo =

* No configuration is needed
* Add the column to show field form in Inbound Messages list (Flamingo is a message storage)
* Enable/Disable Flamingo Addressbook
* Change Flamingo menu name (Ex: Flamingo » Contact Leads)
* Make Inbound Messages to primary menu
* Easy to use and lightweight plugin
* Developer friendly & easy to customize

== Installation ==

1. Upload the `advanced-flamingo` directory to the `/wp-content/plugins/` directory via FTP
2. Alternatively, upload the `advanced-flamingo.zip` file to the 'Plugins->Add New' page in your WordPress admin
area
3. Activate the plugin through the 'Plugins' menu in WordPress

1, 2, 3: You're done!
